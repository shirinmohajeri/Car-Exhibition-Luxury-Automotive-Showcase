## Features

- Responsive modern layout
- Luxury automotive design
- Mobile navigation menu
- Hero section with call-to-action buttons
- Services section
- Car gallery
- About section
- Contact section
- Smooth scrolling
- Modern hover effects

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Installation

1. Download or clone the repository

```bash
git clone https://github.com/your-username/car-exhibition.git
```

2. Open the project folder

3. Run the project by opening:

```bash
jalaseye14.html
```
